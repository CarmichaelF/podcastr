<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/CarmichaelF/podcastr.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
   or
   ```sh
   yarn
   ```
3. Run the project locally using
   ```sh
   npm server
   ```
   or
   ```sh
   yarn server
   ```
   and

   ```sh
   npm dev
   ```
   or
   ```sh
   yarn dev
   ```
